Introduction

In cs124, the second project as a calendar program. Given a year and a month, the program will display a monthly calendar.

Description Calendar project from cs124

The course cs124 was an introduction to programming using C++. One of the assignments was to create a calendar program. You will be programming a Python 3 program to display a monthly calendar based on a given month and year from the user.

Here is a screen shot of the program running.

Your program must:

  • Only except month values from 1 to 12.
  • Only except years from 1753 or later.

Your program will be counting the number of days from Jan 1st, 1753 to the current month and year given by the user:

  • You must create a function called computeOffset() that takes in the month and years and returns the offset value outlined below.
  • You must not use a formula to compute the offset - use loops.
  • Take the number of days from 1st Jan 1753 and return that count modulus 7.
