10 Teach : Team Activity A

Objectives

  • Learn to use different geometries and materials

Overview

The goal for this team activity is to create a 3D clock. The clock should have the following:

  • numbers and/or marks for the hours
  • hour and minute hands (Optional second hand)
  • display the current time.
var d = new Date();

mins = d.getMinutes();
