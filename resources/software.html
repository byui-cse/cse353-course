Verify Python Install¶

Open a command prompt or Mac terminal and type python --version. This should display the current version of Python that you have on your computer. Mac users might need to type python3 --version.

Visual Studio Code¶

During the course, we will be using the text editor Visual Studio Code. It was selected because it's free and can run on Windows, Mac and Linux. You are free to use any editor that you want. However, I will be using VSCode during the course.

For Mac users, if you don't have the latest version of Python, after you install VSCode and try writing your first Python program, VSCode will display this warning. You must install the latest version of Python. Click on the "Download" button. After you download and install Python 3.8+, VSCode will ask you to select the Python version to use - select 3.8+.

Verify Python setup in VSCode¶

  1. Open VSCode and create a Python file with the contents of:

Print('Hello World')

  1. Save this file anywhere on your computer,
  2. In the bottom-left of VSCode, you will see the version of Python that is being used. If it's not the right version (ie., 3.7+), click on it to select a different version of Python.

Installing Python Packages¶

The website https://pypi.org/ contains Python packages.

When you installed Python 3 on your computer, the command pip was also installed. To make sure you have the latest version of pip, here are the commands:

  • On Linux or macOS: pip install -U pip (Note that MAC OS has version 2 of Python pre-installed. Make sure you are running version 3)
  • On Windows: python -m pip install -U pip

At the command prompt, type each of these commands to install the packages required for this course.

pip install opencv-python
pip install numpy
pip install matplotlib
